PVwave procedures for HORACE data

Available as a ZIP file for Download.

The main function to use is GetHoraceData.pro which can extract a time series of data from HORACE.

; PVwave function to extract HORACE data via sockets
;
; Inputs
;
;       LONG   Start   Start time in seconds past midnight
;       LONG   Stop    Stop time in seconds past midnight
;       LONARR params  List of parameters to extract
;
; Outputs
;
;       Returns a FLTARR(number of params,number of data points)
;
; NB. Does not run on VMS systems
;
