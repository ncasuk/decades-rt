1.) Select "View or Tools | Internet Options..." 

2.) Select the "Security" tab 

3.) Select the Zone: "Trusted Sites" 

4.) Click on "Sites..."

5.) Remove tick for "Require server verification (https)"

6.) Type "http://horace" in top box

7.) Click "Add" then "OK"

8.) Set the securety level of this zone: to "Custom" 

9.) Press the "Custom Level" 

10.) In Securety Settings set "Microsoft VM, Java permissions" to "Custom" 

11.) Press the button "Java Custom Settings..." 

12.) Select the "Edit Permissions" tab 

13.) Set everything to "Enable" 

14.) Close the Settings dialogs with "OK" 

15.) Close the Microsoft Internet Explorer
